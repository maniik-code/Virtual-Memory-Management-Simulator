# Virtual-Memory-Management-Simulator

## 📖 Overview
The Virtual Memory Management Simulator is a Python-based project that demonstrates how operating systems efficiently manage memory. It provides a clear understanding of paging, segmentation, and page replacement algorithms through simulation.

---

## 🎯 Objectives
- To understand virtual memory concepts
- To simulate paging and segmentation techniques
- To implement page replacement algorithms
- To analyze page faults and memory utilization

---

## ⚙️ Features
✅ Paging Simulation  
✅ Segmentation Simulation  
✅ FIFO Page Replacement  
✅ LRU Page Replacement  
✅ Optimal Page Replacement  
✅ Page Fault Calculation  
✅ Easy-to-use Command Line Interface  

---

## 🧩 Concepts Covered
- Virtual Memory
- Paging
- Segmentation
- Page Faults
- Memory Allocation
- Replacement Strategies

---

## 🛠️ Technologies Used
- Python 3
- Command Line Interface (CLI)
